# Nuxt3 Sample

Look at the [nuxt 3 documentation](https://v3.nuxtjs.org) to learn more.

## セットアップ

### install

```bash
npm install
cp .env.example .env
```

※ .env 適宜書き換え

## 開発

```bash
npm run dev
```

OR

```bash
make local
```

### for production build

```bash
npm run build
```

### Cloud Run デプロイ

```bash
make deploy
```

公開するとき: https://cloud.google.com/run/docs/securing/managing-access?hl=ja#making_a_service_public
